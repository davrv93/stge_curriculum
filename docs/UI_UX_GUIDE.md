# Guia UI/UX UPeU Silabo AI

## Enfoque visual

La interfaz usa un lenguaje institucional, sobrio y preciso. Evita emojis decorativos, tarjetas infladas y contenedores demasiado anchos.

## Principios

1. Precision antes que espectaculo.
2. Componentes compactos, legibles y funcionales.
3. Colores pastel sobrios como soporte, no como decoracion excesiva.
4. Animaciones suaves de 160 a 220 ms.
5. Iconografia textual o lineal; no emojis tipo chat.

## Tokens principales

- Primario: #1f3a5d
- Fondo: #f6f7f9
- Superficie: #ffffff
- Borde: #e4e8ee
- Texto: #26313d
- Texto secundario: #5f6b7a
- Azul pastel: #dceaf8
- Arena pastel: #f3eadb
- Menta pastel: #dfeee6
- Lavanda pastel: #ebe9f7

## Reglas de componentes

- Cards: padding entre 12 y 18 px, radio 12 px, sombra minima.
- Botones: alto base 38 px, radio 10 px, sin volumen exagerado.
- Formularios: labels arriba, foco con anillo suave.
- Tablas: encabezado sticky, filas compactas, bordes suaves.
- Wizard: pasos pequeños, resumen visible y acciones claras.
- Malla curricular: cursos como chips compactos, color por area de forma sutil.

## Responsive

- Escritorio: sidebar lateral, ancho maximo controlado.
- Tablet: grillas de 2 a 3 columnas segun modulo.
- Movil: sidebar pasa a navegacion superior en grilla, formularios a una columna.
